Tic Tac Toe Game
in Python

This code is a implementation of the classic game Tic Tac Toe using the Python library tkinter. It allows for players to take turns placing their symbol (X or O) on a game board, with the first player to get three of their symbols in a row (horizontally, vertically, or diagonally) declared the winner. The code also handles the case where the game ends in a draw, and includes a reset button to allow for multiple games to be played in a single session.
